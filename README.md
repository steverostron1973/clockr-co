# Clockr.co

Your time and date friend — countdowns, age calculators, time zones and more.

Built with [Astro](https://astro.build), Tailwind CSS, and deployed on Netlify.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Build

```bash
npm run build
npm run preview
```
